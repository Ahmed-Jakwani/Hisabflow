SaaS Manager for Odoo 19

Manage SaaS clients, databases, subscriptions, backups, restores, and logs from inside Odoo.

Main Features

  • Manage SaaS clients and subscription plans.
  • Create client databases from a template database.
  • Check database status and health.
  • Create PostgreSQL backups and restore the latest backup.
  • Suspend and activate client databases.
  • Renew, expire, cancel, and check subscriptions.
  • Track all operations with a dedicated logs model.
  • Arabic translation support.
